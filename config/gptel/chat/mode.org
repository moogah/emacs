* Introduction

Scaffolded stub for =gptel-chat-mode= — the major mode derived from
=org-mode= that owns chat-mode buffers. Real implementation lands in a
later task of the =gptel-chat-mode= OpenSpec change.

See =openspec/changes/gptel-chat-mode/architecture.md= §Components
(subsection =gptel-chat-mode (mode)=) for the full responsibility list:
mode definition, keymap (=C-c C-c=, =C-c C-n= / =C-c C-p=, =C-c C-r=,
display toggle), activation/deactivation hooks, and mode-local vars.
